# 番番记录 · 教师使用说明

番番记录是一款课堂记录与家校反馈微信小程序，支持老师管理学习小组、考勤签到、请假审批和课后反馈。

---

## 一、首次使用：升级教师身份

微信小程序默认登录后是家长身份，需用邀请码升级为教师。

**操作步骤：**

1. 微信搜索"番番记录"小程序，点击**微信登录**
2. 进入底部"我的"Tab → 点击**绑定孩子**
3. 在绑定页面输入你的**教师邀请码**（6 位大写字母），点击确认
4. 提示"已成为老师"，自动跳转首页教师工作台

> ⚠️ 每个邀请码可被多人使用，建议仅分享给本机构老师。

---

## 二、教师工作台（首页）

登录后进入教师首页，顶部显示今日工作和学生总数，下方快捷入口：

| 功能 | 说明 |
|------|------|
| 签到 | 学生上课签到/签退 |
| 发反馈 | 课后学习反馈 |
| 学习小组 | 管理班级和学生 |
| 课表 | 排课管理 |
| 审批 | 请假审批 |

---

## 三、学习小组管理（核心功能）

### 3.1 创建学习小组

1. 首页点击**学习小组** → **新建学习小组**
2. 填写以下信息：
   - **名称**：如"初三数学冲刺班"
   - **年级**：一年级 ~ 高三
   - **学科**：数学 / 物理 / 语文 / 英语 / 化学
3. 点击**保存学习小组**

### 3.2 添加学生

1. 在学习小组卡片中点击展开 → **添加学生**
2. 填写学生信息：
   - **姓名**
   - **性别**：男孩 / 女孩
   - **成绩水平**：下 / 中下 / 中 / 中上 / 好
   - **性格描述**：可选多个标签（最多 8 个），帮助生成个性化反馈
3. 点击**添加学生**，可连续添加

### 3.3 分享邀请码

添加学生后，系统自动生成 6 位邀请码。点击**复制**或**分享**发送给家长。

家长使用此邀请码绑定孩子后，即可查看学习动态、接收反馈。

> ⚠️ 每个学生最多绑定 3 位家长。

## 四、其他功能

### 课表管理
首页 → **课表** → 添加课程安排（日期、时间段、关联学习小组）

### 签到
首页 → **签到** → 选择当日课程 → 对学生签到/签退

### 请假审批
首页 → **审批** → 查看家长请假申请 → 批准/拒绝

### 课后反馈
首页 → **发反馈** → 选择学习小组 → 填写反馈内容（支持图文）

## 五、如何配图

### 需要的截图

| 序号 | 截图内容 | 要点 |
|------|---------|------|
| 1 | 小程序首页（未登录） | 展示"番番记录"品牌和微信登录按钮 |
| 2 | 绑定页面输入教师邀请码 | 展示 6 位码输入框 |
| 3 | 升级成功 / 教师工作台首页 | 展示今日工作面板和快捷入口 |
| 4 | 学习小组管理页面 | 展示已创建的小组列表 |
| 5 | 新建学习小组弹窗 | 展示年级选择、学科选择（5 科） |
| 6 | 添加学生弹窗 | 展示姓名、性别、成绩水平、性格标签 |
| 7 | 学生邀请码分享 | 展示复制/分享按钮 |

### 截图工具

- **微信开发者工具**：打开项目 → 右上角"截屏"按钮（最清晰，推荐）
- **手机截图**：微信打开体验版 → 音量下+电源键截图 → 发送到电脑
- **微信截图快捷键**：PC 微信 `Alt+A`

### 图片处理

- 尺寸建议：750×1334（手机比例），或等比缩放至宽度 600px
- 关键区域可加红色箭头/圈注（用 Snipaste 或 微信自带标注）
- 格式：PNG（清晰）或 JPG（体积小）
- 有隐私信息（学生姓名、邀请码等）记得打码

### 推荐配图排版

```
┌────────────────────────────────┐
│  文字说明                        │
│  [截图]                         │
│  步骤标注                        │
└────────────────────────────────┘
```

每步一张图，图在上文字在下，或左图右文。

## 七、常见问题

**Q：家长说邀请码无效？**
A：检查邀请码是否复制完整（6 位大写字母，无空格），确认该学生不超过 3 位家长绑定。

**Q：如何修改年级/学科？**
A：学习小组管理 → 点击小组卡片上的"编辑"按钮。

**Q：一个老师可以用多个邀请码吗？**
A：不需要。一个邀请码升级后即可使用全部教师功能。
